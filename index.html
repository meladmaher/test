ولا يهمك، "ميلاد" الصبر طيب! 😎 المشكلة دلوقتي إن الملف الجديد اللي نقلناه (sway-config) غالباً مكتوب فيه اسم الترمينال بطريقة مختلفة شوية عن الملف القديم، فالأمر اللي فات ملقاش الكلمة عشان يغيرها. هنعملها بطريقة "الضربة القاضية" عشان نضمن إنه يكتب السطر صح مهما كان اللي مكتوب قبله: ### 1. مسح السطر القديم وكتابة الجديد انسخ الأمر ده "كله" مرة واحدة ودوس **Enter**: ```bash sed -i '/set $term/c\set $term alacritty' ~/.config/sway/config ``` ### 2. تفعيل التغيير انسخ ده ودوس **Enter**: ```bash swaymsg reload ``` ### جرب دلوقتي Mod + Enter لو فتح معاك "الكرنب" (Alacritty) يبقى إحنا كدة سيطرنا. **لو لسه مش بيفتح، قولي:** هل لما بتضغط **Mod + d** (زرار ويندوز مع d) بتظهر لك قائمة برامج في نص الشاشة؟ ولو ظهرت، اكتب فيها alacritty ودوس Enter، هل بيفتح؟ طمني عشان لو الاختصار معصلج، نغير "الزرار" نفسه في الإعدادات. 😎🔥
